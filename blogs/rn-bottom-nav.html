First, let's understand understand how the bottom navigation works. The bottom navigation behaves like a stack of screens, where each tab is an individual screen. When a tab is clicked, the corresponding screen is pushed to the top of the stack. This allows the user to navigate between screens by switching tabs. Imagine a stack of cards where each card represents a screen. When a tab is clicked, the corresponding card is brought to the top of the stack.

Installing the bottom navigation

Creating the bottom navigation

To create the bottom navigation, let's create the bottom navigation component. We are going to named it BottomNav. Here we are going to have the main navigation with the screens we want to display. In the tabBar we are going to call the CustomBottomNav component and pass all the props.

Styling the bottom navigation

To style the bottom navigation, we can use the StyleSheet component from React Native. Also,we are going to create a new component called CustomBottomNav which had been called where we defined the bottom navigation . Now we can define the styles for the navigation bar. In the next step we are going to create a component called CustomMenuIcon to style the icons

Creating custom icons

To create the custom icons, we need to create a new component called CustomMenuIcon. This component will contain the tabs' icons and the animation logic for when the users tap on them. Moreover, We can also define different styles for the active tab to indicate which tab is currently selected. In this case, added animation to the icon when it is tapped and the color when the icons are focused.

Using the bottom navigation

To use the bottom navigation in our app, we simply need to import the BottomNav component and add it to our main app component. We can pass the tab components as children to the NavigationContainer component, and it will take care of rendering the tabs and switching between them.
